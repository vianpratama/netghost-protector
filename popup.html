🛡️ Stealth Mode

Menghapus jejak penjelajahan secara otomatis setiap kali Anda menutup tab atau jendela browser.

  • History & Cache dibersihkan otomatis.
  • Cookie sesi dihapus untuk mencegah tracking.

🔒 Network Hardening

Memperkuat pengaturan browser untuk mencegah kebocoran data identitas di jaringan publik.

  • Mencegah kebocoran DNS (Domain Name System).
  • Mematikan WebRTC agar IP asli Anda tidak bocor.

🌐 Proxy Tunnel

Mengalihkan lalu lintas data melalui server aman (SOCKS5). Admin WiFi hanya akan melihat lalu lintas terenkripsi ke server proxy.

  • Gunakan Host & Port dari penyedia proxy terpercaya.
  • IP asli Anda akan disembunyikan sepenuhnya.

⚠️ Panic Button (Wipe All)

Fitur darurat untuk menghapus SELURUH data browser seketika dan menutup semua tab yang aktif.

Gunakan hanya saat situasi kritis!
